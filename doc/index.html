API reference

В FF реализованы математические и базовые физические сущности, необходимые для работы с real-time графикой и физической симуляцией.
В FF определены макросы для удобства работы с фунциями и методами, показывающие какой параметр является входным (__FF_IN) и выходным (__FF_OUT).
В FF определены общие математические функции, такие как быстрое нахождение квадратного корня, получение модуля числа, сравнение чисел, определение удаленности от нуля, возведение в квадрат, перевод радиан в градусы и наоборот, определение максимального, минимального и среднего в наборе чисел.
Сущность Свойства Методы
Vector2 Поля X, Y Конструктор, инстанцирующий объект со скалярных значений, конструктор копирования; сеттеры и геттеры для каждой компоненты; перегруженные арифметические операторы на скалярные значения (применяют скаляр к каждой компоненте) и операторы сложения и вычитания векторов (покомпонентное вычитание и сложение); перегруженный оператор [] для получения элемента по индексу; функции нахождения проекции и отражения вектора, скалярного произведения, величины и величины в квадрате (норма и норма в квадрате), нормализации вектора.
Vector3 Поля X, Y, Z Совпадает с Vector2, с учетом добавления размерности. Добавлен метод нахождения векторного произведения.
Vector4 Поля X, Y, Z, W Совпадает с Vector3, с учетом добавления размерности.
Matrix2x2 Поля V11, V12, V21, V22 Конструктор, инстанцирующий объект со скалярных значений, конструктор копирования, конструктор инстанцирующий объект с векторов; сеттеры и геттеры для каждой компоненты; перегруженные арифметические операторы на скалярные значения (применяют скаляр к каждому элементу) и операторы сложения, вычитания (покомпонентное вычитание и сложение) и умножения матриц (каждый лемент есть сумма произведение соответствующего элемента строки одной матрицы на столбец другой); перегруженный оператор [] для получения элемента по индексу; функции получения единичной матрицы и нахождения детерминанта.
Matrix3x3 Поля V11, V12, V13, V21, V22, V23, V31, V32, V33 Совпадает с Matrix2x2, с учетом добавления размерности.
Matrix4x4 Поля V11, V12, V13, V14, V21, V22, V23, V24, V31, V32, V33, V34, V41, V42, V43, V44 Совпадает с Matrix3x3, с учетом добавления размерности.
AngleSet3 Поля xRotation, yRotation, zRotation Конструктор, инстанцирующий объект со скалярных значений, конструктор копирования; сеттеры и геттеры для каждой компоненты.
Quaternion Поля X, Y, Z, W Конструктор, инстанцирующий объект со скалярных значений, конструктор копирования; сеттеры и геттеры для матрицы вращения, геттер для вектрной части; перегруженный оператор умножения.
Transforms - Методы вращения матриц по осям, методы масштабирования и сдвига.
Force Direction, Location Конструкторы со скалярных значений и векторов; Сеттеры и геттеры для направления и точки приложения.
MaterialPointBase Mass, Restitution, BoundedSphereRadius, Location, LinearVelocity, LinearAcceleration, ConstantForce, ImpulseForce, isStatic Конструкторы со скалярных значений и векторов; Сеттеры и геттеры для каждого свойства и метод Update для просчёта изменений.
RigidBody Mass, Restitution, BoundedSphereRadius, Location, LinearVelocity, LinearAcceleration, ConstantForce, ImpulseForce, isStatic, CurrentOrientation, RotationalInertia, Torque, AngularVelocity, AngularAcceleration Конструкторы со скалярных значений и векторов; Сеттеры и геттеры для каждого свойства и метод Update для просчёта изменений.
Spring RestLength, ForceConstant, DampeningFactor, PointMass_1, PointMass_2 Конструкторы со скалярных значений и векторов; Сеттеры и геттеры для каждого свойства; метод isDisplaced для отслеживания смещения и метод CalcualteReactions для просчёта реакции.
Spring TotalRows, TotalColumns, TotalSprings, Particles, Springs, Squares, LinearDampeningCoefficient Конструкторы со скалярных значений и векторов; Сеттеры и геттеры для каждого свойства и метод Update для просчёта изменений.
